## Integration Patterns

### 1. Embedded Integration
Seamlessly integrate Corridor's payment infrastructure into your existing platform.

**Use Cases:**
- Add EWA capabilities to existing payroll systems
- Embed social payment features in community apps
- Integrate treasury management into accounting software

**Benefits:**
- White-labeled experience
- Custom branding
- Deep platform integration
- Shared user authentication

### 2. White-Label Solution
Deploy Corridor's complete platform under your brand.

**Use Cases:**
- Launch a complete payment platform
- Offer comprehensive financial services
- Enter new markets quickly

**Benefits:**
- Full platform customization
- Your branding throughout
- Complete feature set
- Dedicated support

### 3. API-Only Integration
Build custom solutions using Corridor's APIs.

**Use Cases:**
- Custom payment flows
- Specialized business logic
- Unique user experiences
- Advanced automation

**Benefits:**
- Maximum flexibility
- Custom development
- Selective feature usage
- Technical control
