# Welcome to Corridor

Corridor is a modern financial platform that enables businesses to send and receive payments globally using traditional currencies and stablecoins.

## What is Corridor?

Corridor combines the best of traditional finance and blockchain technology to provide:

- **Global Payments** - Send money across borders instantly
- **Stablecoin Support** - Use USDC and other stablecoins for faster, cheaper transfers
- **Early Wage Access (EWA)** - Give employees access to earned wages before payday
- **Invoicing** - Create and manage professional invoices
- **Analytics** - Track all your financial activity in real-time

## Key Features

### For Businesses
- Multi-currency wallets
- Mass payouts to employees and contractors
- API access for custom integrations
- Compliance and KYC built-in

### For Developers
- RESTful API with comprehensive documentation
- Webhook support for real-time events
- SDKs for popular programming languages
- Sandbox environment for testing
